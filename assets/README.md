# assets (선택)

이 폴더에 PNG를 넣으면 게임이 **자동으로 그림을 교체**합니다. 없으면 절차적 픽셀 렌더링을 사용합니다.

| 파일명 | 용도 | 권장 사양 |
|--------|------|-----------|
| `bg-hill.png` | 어드벤처/메뉴 배경(언덕·은하수 장면) | 가로로 긴 이미지(예: 1920×1080+), **아빠 캐릭터 없는 버전** 권장 |
| `lyra.png` | 주인공 Lyra 스프라이트 | 배경 투명 PNG, 인물만 잘라낸 세로형, 발끝이 아래 중앙 |

> 캐릭터 디자인 시트에서 Lyra만 잘라내고, 배경 그림에서 아빠를 지운 버전을 각각 저장하면 됩니다.
> 파일을 넣은 뒤 브라우저를 새로고침하면 적용됩니다.
